README.md
A smart, automated standalone desktop application designed to clean up cluttered directories by sorting files into organized subfolders based on their file extensions.
Overview
Managing a crowded Downloads or Desktop folder can be time-consuming. This standalone utility automatically scans a target directory, identifies file types, and systematically transfers them into designated categorical folders, like Documents, Images, or Media. This helps maintain a clean workspace and improves productivity without requiring a Python installation to run.
Key Features
* Standalone Executable: Runs directly as a native desktop application (.exe) with no dependencies or runtimes required.
* Extension Mapping: Groups similar file extensions into standardized categories automatically.
* Safe File Handling: Prevents overwriting by checking if a file with the same name already exists in the destination folder.
Default Category Mapping
Category Folder
	Supported Extensions
 
	Documents
	.pdf, .docx, .txt, .xlsx, .pptx, .csv
	Images
	.jpg, .jpeg, .png, .gif, .svg, .bmp
	Media (Audio/Video)
	.mp3, .mp4, .mkv, .mov, .wav
	Archives
	.zip, .tar, .gz, .rar, .7z
	

How to Use the Application
Since this application has been compiled into a standalone executable, running it is straightforward:
1. Download the file_organizer.exe file from this repository.
2. Double-click the executable to launch the application interface.
3. When prompted, provide or select the comprehensive target directory path you wish to organize.
